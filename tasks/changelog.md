# Changelog

All notable changes to PDF Utilities App will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - Initial Release

### Features

- Convert PDF to DOCX
- Compress PDF files with multiple quality levels
- Merge multiple PDF files
- Split PDF files (every page, custom range, size-based)
- Extract text and images from PDFs
- Convert PDF pages to images (JPG/PNG)
- Password management for password-protected PDFs
- Drag-and-drop file selection
- Batch processing support
- Modern JavaFX user interface
